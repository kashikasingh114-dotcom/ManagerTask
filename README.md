# 🗂️ Task Manager App

A simple and efficient Task Manager Web Application with calendar tracking, progress monitoring, and dark mode support. This project helps users manage tasks in an organized and visual way.

---

## 🚀 Features

- Add new tasks  
- Delete tasks  
- Calendar-based task tracking  
- Progress bar to track completion  
- Dark mode support  
- Clean and simple UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Python
